# Cat-vs-Dog-Image-Classification-using-CNN
This project is a deep learning-based image classification system that distinguishes between cats and dogs using a Convolutional Neural Network (CNN). The trained model is deployed using a Flask web application, allowing users to upload images through a simple frontend interface and receive real-time predictions.

#### The system includes:
A CNN model trained on image data

A Flask backend for inference

A frontend interface for image upload

Static storage for uploaded images

#### Features
Image classification using CNN
Web interface for easy interaction
Real-time prediction
Upload and display images
Confidence-based classification (extendable)

#### Model Details
Framework: TensorFlow / Keras
Input Size: 128x128
Output: Binary Classification (Cat / Dog)
Preprocessing: Image normalization (rescaling)

#### Tech Stack
Python
TensorFlow / Keras
Flask
HTML / CSS
NumPy
